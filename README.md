## Dependencies
Install these three dependencies under the root directory/folder before running the program.
1. Express: ```npm install express```
2. Body-Parser: ```npm install body-parser```
3. Mongoose: ```npm install mongoose```
### Running/Testing
> To run the program, enter this in the terminal under the app directory/folder: ```node app.js```.
